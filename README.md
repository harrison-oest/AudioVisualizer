# AudioVisualizer
This repo contains work for the final project for CSCI-250. 

Components used:
- Raspberry Pi Model 3B+
- GPIO Shield
- Breadboard
- 5V, 20A DC Power Supply
- APA102C RGB LED Strips (4)
- Microphone Sensor
- Temperature Sensor
- Push Button
- IR Receiver and Remote
- Various Jumper Wires
